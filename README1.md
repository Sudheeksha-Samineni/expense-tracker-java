# Expense Tracker (Java)

A simple console-based Expense Tracker application built using Java.

## Features

* Add Expense
* View Expenses
* Show Total Expense
* Category-wise Expense Summary
* Delete Expense
## Technologies Used

* Java
* ArrayList
* HashMap
* OOP Concepts
## How to Run

Compile the program:
javac ExpenseTracker.java

Run the program:
java ExpenseTracker
